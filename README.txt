
- Sistema de gestion de productos

* Descripcion
El sistema de gestion de productos es una aplicacion de linea de comandos que permite gestionar un inventario de productos. 
Las funcionalidades incluyen
1) Agregar productos 
2) Mostrar productos 
3) Actualizar producto 
4) Eliminar producto
5) Buscar producto por ID 
6) Genera un reporte de bajo stock

* Requisitos

- Python 3.x
- SQLite (incluido en Python)

* Instalacion

1. Asegurate de tener Python 3.x instalado en tu sistema. Puedes descargarlo desde 
[python.org](https://www.python.org/downloads/).
2. Descarga este repositorio.

* Ejecucion

1. Abre una terminal o linea de comandos.
2. Navega al directorio donde se encuentra el archivo `sistema de gestion de productos.py`.

3. Ejecuta el siguiente comando para iniciar la aplicaci√≥n:
   python sistema de gestion de productos.py
